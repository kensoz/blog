## 機能

+ 動的および静的リソースの合理的な割り当てる
+ プロキシでの開発とデプロイ。複数のサーバーがある場合は、負荷分散と呼ばれる
+ gzip。js と css などのファイルを圧縮する
+ 設定ファイルでは、変数によってサーバーを選択する
+ httpsの強制設定
+ リクエスト制限
+ アクセス制御
