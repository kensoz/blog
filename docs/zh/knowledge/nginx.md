## 功能

+ 动静结合，合理配置资源
+ 开发常用反向代理，如果是多台服务器就叫负载均衡
+ gzip，压缩js和css文件等
+ 在设定文件，可以根据变量来选择服务器
+ 防盗链
+ 强制配置https
+ 请求限制
+ 访问控制
